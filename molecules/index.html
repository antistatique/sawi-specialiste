flex-grid

Use .flex-grid to display cards in a row. Cards are by default set to a minimum of 250px width. If you want the last one to keep its size, add 2 empty divs at the end of the container.

You can combine the class .flex-grid-[i] where [i] is a number from 1 to 12. It will set the number of items per line, with a minimum of 100px width;
